Google CodeU Chat App

Click here to view the chat app on Github

Through the Google CodeU program, an invite only remote program to strengthen and develop the skills of future software engineers, I helped develop a command line chat app. I added a feature that changed the command line input processing to tokenize input before passing it to a command for execution. I also implemented a feature that saves and restores the current state of the server through serialization. Finally I added a feature that allowed users to have access control depending on their role in the chat.
