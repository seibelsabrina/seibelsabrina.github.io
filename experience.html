Internships

Summer 2019

I was on Google's Android Infrastructure Team where I worked on two main projects related to optimizations and efficiency.

The initial project that I tackled this summer was optimizing Java Protocol Buffer runtime. In order to make this process safe, I made major refactorings to the codebase using static analysis.

Another project that I worked on was creating a tool to analyze generated source code.The end goal of this tool was to decrease APK size for Google applications. By identifying unnecessary generated code, we are able to cut out superfluous weight from applications.

Summer 2018

I was on Google's Java Static Analysis team where I primarily worked on Error Prone. Error Prone is an open-source compiler plugin designed to find and eliminate bugs in Java code, and is used within Google's Java build system.

For Error Prone, I worked on the compiler itself, where I helped enhance its ability to detect bugs, fix bugs, and enforce good code practices. To do this, I wrote checks that look for and match specific patterns within the AST. When a match is identified, the check replaces the incorrect code with valid alternative.

A specific check I wrote pointed out a common developer mistake, passing an Integer into Math.round(), inadvertently truncates the number.

Aside from Error Prone, I contributed to a Build Diagnostics tool designed to increase developer efficiency!
